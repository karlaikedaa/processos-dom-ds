Design: Estrutura de Blocos da Tela Tarefas

Seção 1 - Organização geral da página

Estrutura de 3 Blocos
📍 BLOCO 1: Header (direto no fundo cinza)
• Background: transparent (sem card branco)
• Padding: 0 (sem padding extra)
• Margin-bottom: 16px
• Contém: Título "Tarefas" + Descrição + Botões "Apontar" e "Nova tarefa"
📍 BLOCO 2: Filtros (card branco separado)
• Background: #ffffff
• Border: 1px solid #e5e5e5
• Border-radius: 8px
• Padding: 16px 24px (vertical 16px, horizontal 24px)
• Margin-bottom: 16px
• Contém: Campo de busca + 4 dropdowns com setas
📍 BLOCO 3.1: Header de Abas
• Background: #ffffff
• Padding: 16px 24px
• Border-bottom: 1px solid #e5e5e5
• Contém: 4 abas + 2 botões de ação (Exportar, Filtros Avançados)
• ⚠️ SEM botões duplicados abaixo
📍 BLOCO 3.2: Conteúdo das Abas
• Background: #ffffff (herda do bloco 3)
• Padding: 0 (sem padding extra no wrapper)
• IMPORTANTE: O conteúdo das views (Kanban, Lista, Calendário, Fluxo) renderiza AQUI DENTRO
• Cada view gerencia seu próprio padding interno

📋 Hierarquia JSX/React

<div className="flex flex-col h-full" style={{ background: '#f6f6f6' }}>

  {/* BLOCO 1: Header - direto no fundo cinza */}
  <div className="px-6 pt-6 pb-4">
    <div className="flex items-center justify-between">
      <h1>Tarefas</h1>
      <div>Botões Apontar + Nova tarefa</div>
    </div>
  </div>

  {/* BLOCO 2: Filtros - card branco separado */}
  <div className="mx-6" style={{ marginTop: '16px' }}>
    <div className="bg-white rounded-lg px-6 py-4 border">
      Busca + Dropdowns com setas
    </div>
  </div>

  {/* BLOCO 3: Tabs + Conteúdo - card branco separado */}
  <div className="mx-6 mt-4">
    <div className="bg-white rounded-lg border">

      {/* BLOCO 3.1: Header de Abas */}
      <div className="px-6 py-4 border-b">
        <div>4 Abas + 2 Botões de ação</div>
      </div>

      {/* BLOCO 3.2: Conteúdo das Abas */}
      <div className="flex-1 flex overflow-hidden">
        {viewMode === 'kanban' && <KanbanView />}
        {viewMode === 'lista' && <ListaView />}
        {viewMode === 'calendario' && <CalendarioView />}
        {viewMode === 'fluxo' && <FluxoView />}
      </div>

    </div>
  </div>

</div>
⚠️ Mudança Crítica

A principal mudança é remover o wrapper extra que estava criando os botões duplicados e reorganizar para que:
• Bloco 3.1 = Header com abas E botões (apenas 1x)
• Bloco 3.2 = Conteúdo das views (sem wrapper intermediário)
