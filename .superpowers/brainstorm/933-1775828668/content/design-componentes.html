🔽 1. Dropdowns com Setas (ChevronDown)

❌ Atual (sem seta visível)

Usa <select> nativo sem customização

✓ Corrigido (com ChevronDown)

Wrapper relativo + ícone absoluto + appearance: none

Implementação JSX: 
<div className="relative" style={{ minWidth: '150px' }}>
  <select
    className="w-full rounded-md px-3 py-2 outline-none"
    style={{
      border: '1px solid var(--border)',
      background: 'white',
      fontSize: 'var(--text-label)',
      paddingRight: '36px',
      appearance: 'none',
    }}
  >
    <option>Empresas: Todos</option>
  </select>
  <ChevronDown
    size={14}
    className="absolute right-3 top-1/2 -translate-y-1/2 pointer-events-none"
    style={{ color: 'var(--muted-foreground)' }}
  />
</div>

CSS crítico: appearance: none remove a seta nativa, padding-right: 36px evita sobreposição do texto, pointer-events: none no ícone permite clicar no select
